# User Guide: Advanced Todo Features

## Overview

This guide explains how to use the advanced features of the Todo application, including priorities, tags, search and filter, sort, recurring tasks, and due dates & reminders.

## Getting Started

After logging in to the application, you'll see your task list with new advanced features available.

## Setting Task Priority and Tags

### Setting Priority
1. When creating a new task, select the priority level:
   - **High**: For urgent tasks that need immediate attention
   - **Medium**: For tasks of normal importance
   - **Low**: For tasks that can be deferred

2. When editing an existing task, you can change its priority level using the dropdown menu.

### Adding Tags
1. When creating or editing a task, you can add tags by entering them in the tags field, separated by commas.
2. Tags help you categorize and group related tasks (e.g., "work", "personal", "urgent").
3. You can add up to 10 tags per task.

## Searching and Filtering Tasks

### Search
1. Use the search bar at the top of the task list to find tasks by keyword.
2. The search looks through task titles and descriptions.
3. Results update in real-time as you type.

### Filters
1. Click the "Filters" button to expand advanced filtering options.
2. You can filter by:
   - **Status**: All, Pending, Completed
   - **Priority**: All, Low, Medium, High
   - **Tags**: Select specific tags to filter by
   - **Due Date Range**: Specify a date range to filter tasks

## Sorting Tasks

1. Use the "Sort by" dropdown to choose how tasks are ordered:
   - **Created Date**: Sort by when tasks were created
   - **Due Date**: Sort by task due dates
   - **Priority**: Sort by priority level (high to low)
   - **Title**: Sort alphabetically by title

2. Toggle between ascending and descending order using the sort direction button.

## Creating Recurring Tasks

1. When creating a task, enable the "Recurring" option.
2. Choose the recurrence pattern:
   - **Daily**: Every day
   - **Weekly**: Every week on specific days
   - **Monthly**: Every month on a specific day
   - **Yearly**: Every year on a specific date
3. Set the interval (e.g., every 2 weeks) if needed.
4. Optionally set an end date or occurrence limit.

When a recurring task is completed, the next occurrence is automatically created based on your pattern.

## Setting Due Dates and Reminders

### Due Dates
1. When creating or editing a task, click the calendar icon to set a due date.
2. Select the date and time for your task deadline.
3. Tasks with approaching due dates will be highlighted.
4. Overdue tasks will be marked with a special indicator.

### Reminders
1. Enable the reminder toggle when creating or editing a task.
2. Set the reminder time using the date/time picker.
3. The reminder time should be before the due date.
4. You'll receive a notification at the reminder time.

## Mobile Usage Tips

1. Swipe left on a task to quickly mark it as complete
2. Tap and hold on a task to bring up quick actions
3. Use the floating action button to quickly add a new task
4. Pinch to zoom on the calendar view for due dates

## Troubleshooting

### Task Not Appearing After Creation
- Check that the task was created successfully by looking for a confirmation message
- Verify that your filters aren't hiding the task
- Refresh the page if needed

### Reminder Not Working
- Ensure notifications are enabled in your browser/device settings
- Check that the reminder time is set correctly
- Verify that the due date is in the future

### Search Not Finding Expected Results
- Check spelling in your search query
- Remember that search is case-insensitive
- Try using broader search terms

## Keyboard Shortcuts

- `Ctrl + N` (or `Cmd + N`): Create new task
- `Ctrl + F` (or `Cmd + F`): Focus on search bar
- `Esc`: Close modals or clear search
- `Enter`: Submit forms or confirm actions

## Privacy and Security

- All your tasks and data are securely stored and encrypted
- Only you have access to your task list
- Your data is never shared with third parties
- You can export or delete your data at any time in Account Settings
